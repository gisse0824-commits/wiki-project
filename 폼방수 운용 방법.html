🚒 폼방수 운용 매뉴얼

소방차 폼방수 종류와 단계별 운용 절차임

💧
선 물방수 후 폼투입
물방수를 먼저 하다가 폼을 투입하는 방법임. 일반화재에서 물로 진압하다가 유류 등 특수화재시 폼을 조합함
📋
특장업체 매뉴얼 기본형
최신 특장차 기준, 매뉴얼대로 하는 폼방수임. 폼전용 배관과 밸브/조절장치 활용함 기본형
🔫
관창 직결형 (응용형)
폼액과 물을 관창에서 직접 조합하는 방식임. 폼 관창, 피스톨 관창 등으로 거품 생성함. 현장 상황에 따라 손쉽게 사용 가능함
📏
포스 구경별 폼방수
관창 구경(40mm, 65mm)에 따라 폼 투입량/밸브 설정을 다르게 함. 각 사이즈별 밸브 조절 필요함
📋 특장업체 매뉴얼 폼방수 (폼 메인방식)
1
PTO 연결, 펌프 가동
소방차의 동력인출장치(PTO)를 연결하고 펌프를 가동시킴
2
폼 메인 밸브 개방
폼액이 흐를 수 있도록 폼 메인 밸브를 열 것
3
송수(물공급) 밸브 개방
물 공급을 위한 송수 밸브를 개방할 것
4
폼 조절 밸브 설정 (관창 구경별)
관창 크기에 맞춰 폼 조절 밸브를 설정할 것
🔸 40mm 관창: 130에 맞춤
🔸 65mm 관창: 330~562에 맞춤 (제조업체 매뉴얼 참고)
5
방수 시작
폼액과 물이 희석되어 방수구로 거품이 방수됨
6
RPM(엔진회전수) 조절
필요시 폼 형성력 및 사거리를 조절할 것
7
작업 종료 후 폼 메인 밸브 닫기, 세척
작업이 끝나면 폼 메인 밸브를 닫고 배관을 세척할 것
💧 선 물방수 후 폼투입
1
화재진압 초기에는 물만 방수
일반 화재 진압 시 물만 사용하여 방수를 시작할 것
2
폼 필요 시, 폼 메인 밸브와 폼 조절 밸브 개방
유류화재 등 폼이 필요한 상황에서 폼 관련 밸브들을 열 것
3
폼액이 물과 혼합돼 관창에서 거품 발생
물과 폼액이 혼합되어 관창에서 거품이 생성됨
4
화재종류 및 상황에 따라 폼방수 종료 후, 세척 실시
진압 완료 후 배관 세척을 반드시 실시할 것
🔫 관창 직결형 (응용형) 폼방수
1
폼 전용 관창/피스톨 관창 연결
폼 전용 관창 또는 피스톨 관창을 호스에 연결할 것
2
폼액 통을 관창에 직접 연결
폼액 통을 관창에 직접 연결하여 준비할 것
3
물 공급 밸브 개방 후, 폼액과 물이 관창에서 결합
물 공급 밸브를 열면 폼액과 물이 관창에서 자동으로 혼합됨
4
관창에서 바로 거품 생성, 화재 진압
관창에서 즉시 거품이 생성되어 화재 진압을 시작할 것
📏 포스 구경별 (관창 사이즈별) 폼방수
1
관창 사이즈 확인 (40mm, 65mm 등)
사용할 관창의 구경을 확인할 것
2
폼 조절 밸브를 각 사이즈 매뉴얼에 맞게 설정
관창 사이즈에 따라 폼 조절 밸브를 정확히 설정할 것
🔸 40mm: 130
🔸 65mm: 330~562 (매뉴얼 참고)
3
폼 메인과 송수 밸브 개방
폼 메인 밸브와 송수 밸브를 개방할 것
4
필요 폼양 조절 후 방수 시작
현장 상황에 맞춰 폼 양을 조절하고 방수를 시작할 것
5
작업 종료 후, 각 밸브 닫고 배관 세척 실시
모든 밸브를 닫고 배관 내부를 깨끗이 세척할 것
🧼 폼방수 후 세척방법
1
폼 메인 밸브 닫음
폼액 공급을 차단할 것
2
송수 밸브는 연 상태 유지
물이 계속 흐르도록 유지할 것
3
폼 조절 밸브도 연 상태에서 물로 배관 내 폼 세척
물로 배관 내부의 폼액을 깨끗이 씻어낼 것
4
모든 밸브 닫고 장비 정리
세척 완료 후 모든 밸브를 닫고 정리할 것
💡 실무 포인트
⚠️
차량별 매뉴얼 필수 확인
차량 모델·관창 종류에 따라 세부 밸브명·설정값이 다를 수 있으므로 반드시 차량별 매뉴얼을 참고할 것
🔄
밸브 개폐 순서 및 세척 필수
작업 전후 밸브 개폐 순서를 정확히 지키고, 세척을 반드시 실시할 것
💧
용수 구분 고지
폼방수 사용 후 물공급 다시 할 때는 식수용·생활용과 구분 고지할 것
